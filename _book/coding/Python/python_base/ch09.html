ch09. 常用 Python 标准库

（九）常用 Python 标准库

Python奉行“内置电池”的哲学，内置了大量、全面、高效和跨平台的标准库，提供了非常丰富的功能，能够满足大多数常见的需求。并且很多都是用C语言编写的，执行速度也很快。 我们应该尽量多学习这些标准库，理解它们、掌握它们然后使用它们，避免重复造轮子，增强我们的开发能力，提高开发速度。

本章，我们将介绍以下的模块：

  • os
  • sys
  • subprocess
  • random
  • bisect
  • hashlib
  • queue
  • fileinput
  • shutil
  • zipfile
  • tarfile
  • getpass
  • shelve
  • json
  • pickle
  • time
  • datetime
  • timeit
  • logging
  • re
